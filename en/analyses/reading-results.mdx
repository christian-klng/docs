After an analysis completes, you receive detailed results for each document.

## Results overview

The overview shows all analyzed documents with:

- **Total score:** Sum of all points achieved
- **Maximum score:** Maximum achievable points
- **Percentage:** Ratio of achieved to possible points
- **Status:** Color coding (green/yellow/red)

## Detailed results per document

Click on a document to see individual evaluations:

### Score per step

For each evaluation step you'll see:

| Element | Description |
|---------|-------------|
| **Score** | Points achieved |
| **Maximum** | Maximum achievable points |
| **Status** | Passed / Partial / Failed |

### Strengths

A list of positive aspects the AI identified in the document. Example:

- "The objective is clearly and measurably formulated"
- "The timeline contains realistic milestones"

### Weaknesses

Areas with improvement potential:

- "Resource planning is completely missing"
- "Risks are not addressed"

### Feedback

Detailed text feedback from the AI with specific improvement suggestions.

## Status meanings

| Status | Color | Meaning |
|--------|-------|---------|
| **Passed** | Green | Criterion fully met |
| **Partial** | Yellow | Criterion partially met |
| **Failed** | Red | Criterion not met |

## Exporting results

You can export results as a report:

1. Open the analysis results
2. Click **Export**
3. Choose the format (PDF or XLSX)
4. The report is downloaded

## Comparing runs

If you've run an analysis multiple times:

1. Open the analysis
2. Select the desired run from the dropdown
3. Compare results from different points in time

<Tip>
Use the comparison feature to measure progress after revising documents.
</Tip>
